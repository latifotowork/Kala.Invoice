📊 Import dari Google Sheets (via Apps Script)

ℹ️ Import langsung dari Google Sheet yang terhubung via Apps Script.
Pastikan kamu sudah mengisi dan men-test URL Apps Script di bagian ☁️ Sinkronisasi Multi-Device di bawah.
Import ini akan mengambil data klien & item dari sheet — tidak butuh URL publish CSV terpisah.

☁️ Sinkronisasi Multi-Device

○ Sync nonaktif
⚠️ Data di web masih kosong. Paste URL Apps Script kamu di bawah lalu klik "⬇️ Ambil Data dari Cloud" untuk menarik semua data (settings, dokumen, klien, item) ke device ini.
Cara setup (5 menit):
1. Download file Kala Project_apps_script.gs dari link setup
2. Buka script.google.com → New Project → paste kodenya
3. Klik Deploy → New Deployment → Web App
4. Execute as: Me · Who has access: Anyone → Deploy
5. Copy URL deployment → paste di kolom bawah ini
💡 Setelah setup, setiap simpan/hapus otomatis tersinkron ke semua device.

📂 Import CSV Offline (dari file lokal)

Import langsung dari file .csv di device kamu: Format kolom Klien: Nama | Perusahaan | Telepon | Email | Alamat
Format kolom Item: Nama Item | Kategori | Harga | Keterangan | Satuan
Baris pertama (header) akan otomatis dilewati.

🕐 Import Invoice / Dokumen Historis

Untuk memasukkan invoice lama dari bulan-bulan sebelumnya: Upload file .csv dengan format berikut (baris pertama = header, dilewati otomatis):

No. Dokumen, Tipe, Tanggal, Klien, Perusahaan, Telepon, Email, Alamat Klien, Nama Item, Qty, Harga Satuan, Keterangan Item, Diskon(%), PPN(%), Catatan, Status, Bank, No. Rekening, Atas Nama

💡 Tips: Jika satu invoice punya banyak item, buat baris terpisah untuk setiap item dengan No. Dokumen yang sama — sistem akan menggabungkannya otomatis.
Kolom Tipe isi dengan: invoice, quotation, proposal, atau receipt
Kolom Status isi dengan: paid, unpaid, sent, atau draft
